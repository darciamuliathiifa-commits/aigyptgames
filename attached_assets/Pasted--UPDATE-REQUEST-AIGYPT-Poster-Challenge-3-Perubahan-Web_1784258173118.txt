## Perubahan 3 — Card Contoh Karya (dikelola dari Admin Panel)

Tambah section "Contoh Karyanya Kayak Gini 👇" yang muncul SEBELUM user mulai misi:

- **Lokasi tampil**: di landing page (setelah langkah misi, sebelum CTA "Ikutan Sekarang") dan di halaman Misi (di atas step 1).
- **Bentuk**: carousel/scroll horizontal berisi card contoh poster — gambar poster rasio 4:5, rounded-2xl, glow ungu halus, caption kecil di bawahnya (misal nama anomali yang dipakai). Bisa di-swipe di mobile.
- Kalau belum ada contoh yang di-upload, section ini disembunyikan total (jangan tampil kosong).

**Backend**:

```sql
create table example_posters (
  id uuid primary key default gen_random_uuid(),
  image_url text not null,
  caption text,
  sort_order int default 0,
  active boolean default true,
  created_at timestamptz default now()
);
```

- RLS: select anon hanya baris `active = true`; insert/update/delete hanya service role.
- Simpan gambarnya di bucket storage yang sudah ada.
- Tulis SQL-nya sebagai file migration terpisah (`supabase/migration_v2.sql`) — gue jalankan manual, JANGAN otomatis.

**Admin Panel** — tambah tab baru "Contoh Karya":

- Upload gambar (preview sebelum simpan), field caption opsional, drag/tombol untuk atur urutan (sort_order), toggle aktif/nonaktif, tombol hapus.
